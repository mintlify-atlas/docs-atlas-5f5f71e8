Reze Studio brings professional MMD animation editing to the web. Powered by WebGPU and built on [reze-engine](https://github.com/AmyangXYZ/reze-engine), it delivers a full timeline, dopesheet, and per-channel Bézier curve editor for hand-keying `.vmd` clips — on any device from an iPad to a gaming laptop.

## What makes Reze Studio different

Reze Studio is a focused, cross-platform animation tool built to do the animation-editing job exceptionally well. It is not a full MMD replacement and it is not trying to be Maya or Blender — it is the dedicated curve editor that MMD animators have always wanted, freed from the Windows-only desktop install.

## Get started in four steps

<Steps>
  <Step title="Open the editor">
    Navigate to [reze.studio](https://reze.studio) — a default model and sample clip load automatically so you can start exploring immediately.
  </Step>
  <Step title="Load your own model (optional)">
    Go to **File → Load PMX folder…** and select the folder containing your `.pmx` file. Textures must sit next to the model file.
  </Step>
  <Step title="Load or create a clip">
    Use **File → Load VMD…** to import an existing animation, or **File → New** to start a fresh clip on the loaded model.
  </Step>
  <Step title="Edit and export">
    Pick bones, pose them, shape the curves, then **File → Export VMD…** to save. There is no server — nothing leaves your browser.
  </Step>
</Steps>

<Note>
  Reze Studio requires a browser with WebGPU support. Chrome 113+, Edge 113+, and Safari 18+ are fully supported. Firefox users should enable the `dom.webgpu.enabled` flag in `about:config`.
</Note>
